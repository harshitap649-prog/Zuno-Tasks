# üéØ How to Enable CPALead Watch Ad & Banner Ads

This guide will help you enable both the **Watch Ad function** and **Banner Ads** from CPALead.

## üìã Prerequisites

1. ‚úÖ **CPALead Account**: You need a verified CPALead account
2. ‚úÖ **Admin Access**: You must be logged in as an admin
3. ‚úÖ **CPALead Offerwall**: You need to get your Offerwall Direct Link URL from CPALead

---

## üöÄ Step-by-Step Setup

### **Step 1: Get Your CPALead Offerwall Direct Link**

1. **Login to CPALead Dashboard**
   - Go to: https://www.cpalead.com/
   - Login with your publisher account

2. **Navigate to Reward Tools**
   - Click on **"Reward Tools"** in the left sidebar
   - Click on **"Your Offerwall"**

3. **Get the Direct Link**
   - Click the **"GET CODE"** button
   - Find the **"Direct Link"** section
   - Copy the Direct Link URL
   - Example format: `https://zwidgetbv3dft.xyz/list/gMez9KHT`

---

### **Step 2: Configure in Admin Panel**

1. **Go to Admin Panel**
   - Navigate to your website
   - Login as admin
   - Click on **"Help & Support"** (or **"Admin"**) in the top navigation

2. **Open Offers Tab**
   - Click on the **"Offers"** tab in the admin panel

3. **Open CPALead Configuration**
   - Find the **"CPALead VIP"** button/section
   - Click on it to open the configuration form

4. **Enter Your Offerwall Direct Link**
   - In the **"CPALead Direct Link URL"** field
   - Paste the Direct Link URL you copied from Step 1
   - Example: `https://zwidgetbv3dft.xyz/list/gMez9KHT`

5. **Save Configuration**
   - Click the **"Enable CPALead Tools"** button
   - You should see a success message

---

## ‚úÖ What Gets Enabled Automatically

Once you save the CPALead Offerwall Direct Link URL, **both features are automatically enabled**:

### 1. **Banner Ads** üì∫
- **Location**: Bottom of every page (fixed at bottom)
- **What it shows**: CPALead Offerwall in a banner format
- **Visibility**: Only shows if CPALead is configured
- **Auto-updates**: Uses the same URL you configured

### 2. **Watch Ad Function** üé¨
- **Location**: Dashboard page ‚Üí "Watch Ad" button
- **What it shows**: CPALead Offerwall in a modal
- **Functionality**: Users complete offers to earn points
- **Auto-updates**: Uses the same URL you configured

---

## üîç How to Verify It's Working

### **Check Banner Ads:**
1. Go to any page (Dashboard, Tasks, Wallet, etc.)
2. Scroll to the bottom of the page
3. You should see a banner with CPALead offers
4. If you don't see it, make sure CPALead is configured correctly

### **Check Watch Ad Function:**
1. Go to the **Dashboard** page
2. Look for the **"Watch Ad"** button/card
3. Click on it
4. A modal should open with CPALead Offerwall
5. Users can complete offers to earn points

- ‚úÖ Check if CPALead Offerwall URL is saved in admin panel

- ‚úÖ Verify the Direct Link URL format is correct

### **Both Not Working:**
- ‚úÖ Double-check the Direct Link URL in admin panel
- ‚úÖ Make sure you clicked "Enable CPALead Tools" button

## üìù Important Notes

1. **Single Configuration**: You only need to configure the Offerwall Direct Link URL once. Both banner ads and watch ad function use the same URL.

2. **No Additional Setup**: Once you configure the Offerwall URL, both features work automatically. No separate configuration needed.

3. **URL Format**: Make sure you're using the **Direct Link** URL, not the embed code or widget code.

4. **User Tracking**: The system automatically appends user IDs to track conversions, so CPALead can credit your account properly.
