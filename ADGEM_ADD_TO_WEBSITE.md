## üìã **Step-by-Step Guide:**

### **Step 1: Get Your AdGem API Key/URL**

1. **In AdGem Dashboard:**
   - You're on the "Properties & Apps" page
   - Find your "zunotasks" property in the table
   - Click the **"Options"** button (purple button with dropdown arrow)
   - A menu will appear

2. **What to Look For:**
   - **Option 1:** Look for "API Key" or "Publisher ID"
   - **Option 2:** Look for "Offerwall URL" or "Integration URL"
   - **Option 3:** Look for "Direct Link" or "Integration Details"

3. **Copy One of These:**
   - **API Key/Publisher ID:** A long string of characters
   - **Offerwall URL:** A URL like `https://www.adgem.com/offers?pub=YOUR_KEY&sub={USER_ID}`
   - **Direct Link URL:** Similar format

**Save this information - you'll need it in Step 2!**

---

### **Step 2: Configure in Your Admin Panel**

1. **Go to Your Website:**
   - Open your website in a new tab
   - Log in as **admin**

2. **Navigate to Admin Panel:**
   - Click on **"Admin"** in the navigation menu
   - Or go to: `https://your-domain.com/admin`

3. **Go to Offers Tab:**
   - In Admin Panel, click on **"Offers"** tab
   - Scroll down to find AdGem section

4. **Find AdGem Button:**
   - Look for **"AdGem (All Tasks)"** button
   - It should be a green button
   - Click on it to expand the configuration

5. **Enter Your AdGem Details:**
   
   **If you have an Offerwall URL:**
   - Paste it in the **"AdGem Offerwall URL"** field
   - Format: `https://www.adgem.com/offers?pub=YOUR_KEY&sub={USER_ID}`
   - Make sure it has `{USER_ID}` placeholder (system will replace it)
   
   **If you have an API Key:**
   - Paste it in the **"AdGem API Key"** field
   - Just the key, no URL needed

6. **Save Configuration:**
   - Click **"Enable AdGem"** button
   - Wait for confirmation message
   - You should see: "‚úÖ AdGem is ready!"

---

### **Step 3: Verify It's Working**

1. **Go to Tasks Page:**
   - Navigate to **"Tasks"** page on your website
   - (Not as admin, as a regular user)

2. **Check for Music Tasks:**
   - Look for **"Listen to Music"** button in category tabs
   - It should appear automatically when AdGem is configured
   - Click on it

3. **See AdGem Offers:**
   - You should see **"AdGem Music"** section
   - Click **"Listen to Music"** button
   - AdGem offerwall should open

4. **Test It:**
   - Complete a music/video offer
   - Points should be awarded automatically
   - Check your points balance

- Keep `{USER_ID}` as is (system replaces it)

**Option B: If you have just API Key:**
- Just paste the API key in the "API Key" field
- System will construct the URL automatically

## ‚ùì **Troubleshooting:**

### "I can't find the API key in Options menu"
- **Solution:** Click on the property name "zunotasks" - it might open details
- Or check "Integration Documentation" section
- Or look for "Settings" or "API" in the sidebar

### "Options button doesn't show API key"
- **Solution:** The API key might be in a different section
- Try clicking on the property ID (31438) or name
- Check "Integration" or "API" tabs

### "Listen to Music button doesn't appear"
- **Solution:** 
  1. Make sure you saved AdGem configuration
  2. Refresh the Tasks page
  3. Check Admin Panel to verify AdGem is enabled
  4. Clear browser cache if needed

### "Offers not showing"
- **Solution:**
  1. Wait a few minutes (AdGem needs to sync)
  2. Check if your property is approved (status should be "Active" not "New")
  3. Verify API key is correct
  4. Try refreshing the page
