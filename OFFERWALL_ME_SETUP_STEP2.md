# Step 2: Get Your API Key / Integration Code

## What to Look For:

Now that you're in Publisher Panel, you need to find:

### Option 1: API Key / Publisher ID
- Look for menu items like:
  - **"API"** or **"API Keys"**
  - **"Integration"** or **"Get Started"**
  - **"Settings"** → **"API"**
  - **"Publisher ID"** or **"User ID"**

### Option 2: Offerwall URL / Iframe Code
- Look for:
  - **"Offerwall"** or **"Offers"**
  - **"Integration Tools"**
  - **"Embed Code"** or **"Widget"**
  - **"Iframe URL"**

### Option 3: Available Offers Section
- Look for:
  - **"Offers"** or **"Available Offers"**
  - **"Tasks"** (you saw this in the menu)
  - **"Offer List"**

---

## What Information You Need:

You're looking for ONE of these:

1. **API Key / Publisher ID** (best option)
   - Looks like: `abc123xyz` or `PUB123456`
   - Used for API integration

2. **Offerwall URL** (alternative)
   - Looks like: `https://offerwall.me/offerwall?user_id=XXX&api_key=YYY`
   - Used for iframe embed

3. **Your User ID / Publisher ID**
   - Usually shown in your profile/dashboard
   - Combined with API key for full integration

---

## Where to Find It:

### Check These Places:

1. **Dashboard** (first page you see)
   - Might show your Publisher ID at top
   - Look for "Your ID" or similar

2. **Settings** menu
   - Click "Settings" → Look for "API" or "Integration"

3. **API / Integration** menu
   - Direct link in sidebar or menu
   - Shows your API keys and codes

4. **Get Started** or **Quick Start** section
   - Usually has integration guide
   - Shows API key there

5. **Profile** or **Account** section
   - Your Publisher ID might be here

---

## What You Should See:

Once you find it, you'll see something like:

```
API Key: abc123xyz456
OR
Publisher ID: PUB123456
OR
Offerwall URL: https://offerwall.me/offerwall?user_id=123&api_key=abc...
```

---

## Action Required:

1. **Navigate** through the Publisher Panel menu
2. **Look for** API/Integration/Settings sections
3. **Find** your API Key or Publisher ID
4. **Copy it** - you'll need it in the next step
